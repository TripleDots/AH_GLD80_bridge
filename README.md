Allen & Heath GLD 80 bridge

A custom bridge to bring more features to MIDI with MCU.
This makes the desk a bit more studio ready.
Keep in mind that the faders are only 7 bit (127 steps). So you won't be doing precize mixes and/or automations with it.
But it works great for controlling virtual instruments.

For REAPER users, it comes with a companion app.
This allows a deeper integration, like being able to color the tracks based on DAW track colors.
There aren't much DAWs that allow you to do that. So for other DAWs, you either have to color it on the desk or leave it as default.

I have only tested it on Windows 11 with REAPER and Davinci Resolve.

If you want to vibe code any changes, just upload the .zip to ChatGPT. You'll need a Plus account or else you'll have to constantly add the new code yourself.
